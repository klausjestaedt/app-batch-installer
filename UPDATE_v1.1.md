# [apps.sh]

## Was ist neu

Für Debian basierte Applisten können in einer Zeile nun auch Repositories angegeben werden.
Statt dem Appnamen kommt dann repo: gefolgt von Repository.
Beispiel für das Programm: fastfetch

Die Datenzeile und danach die Zeile für das Programm.
Ab dem Semikolon ist alles optionale Beschreibung:

repo:ppa:zhangsongcui3371/fastfetch;Repo for fastfetch
fastfetch

## Whats new
For Debian-based app lists, repositories can now also be specified on a single line.
Instead of the app name, use repo: followed by the repository.
Example for the program: fastfetch

The data line followed by the line for the program.
Everything after the semicolon is an optional description:

repo:ppa:zhangsongcui3371/fastfetch;Repo for fastfetch
fastfetch
